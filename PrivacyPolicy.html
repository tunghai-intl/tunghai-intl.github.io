一、政策說明

歡迎您瀏覽本網站。本公司重視您的個人資料保護，並依相關法令規定妥善處理及保護您提供的資訊。

本政策說明您於使用本網站時，我們可能蒐集之資訊、使用方式及相關權利。

二、資料蒐集項目

當您透過本網站聯絡表單與我們聯繫時，可能提供以下資訊：

  • 姓名
  • 公司名稱
  • 電子郵件地址
  • 聯絡電話
  • 詢問內容及其他您主動提供之資訊

此外，網站系統可能自動記錄：

  • IP 位址
  • 瀏覽器類型
  • 裝置資訊
  • 存取時間及瀏覽紀錄

作為網站維護、安全管理及服務改善之用途。

三、資料使用目的

我們蒐集之資料將用於：

回覆您的詢問與需求
提供產品或服務相關資訊
商務聯繫與合作洽談
網站安全維護及效能優化
法令規定之必要用途

四、第三方服務

本網站使用第三方服務提供網站運作及資料傳輸功能，包括但不限於：

GitHub Pages

本網站部分內容透過 GitHub Pages 提供網站託管服務。

Cloudflare

本網站使用 Cloudflare 提供內容傳遞網路（CDN）、網站效能優化及安全防護服務。

Web3Forms

當您提交聯絡表單時，相關資料將透過 Web3Forms 服務進行傳輸並寄送至本公司指定電子郵件信箱，以便處理您的詢問。

上述第三方服務可能依其自身隱私政策處理相關技術性資訊，建議您參閱其官方說明。

五、求職資訊

本網站所提供之職缺資訊或「加入我們」頁面，可能連結至第三方求職平台（如 104 人力銀行）。

當您點擊連結並前往該平台後，相關個人資料之蒐集、處理及利用，將依該平台之隱私權政策及服務條款辦理，本公司不直接蒐集或保存您的履歷資料。

六、資料分享與揭露

除法律規定、主管機關要求或取得您的同意外，本公司不會將您的個人資料出售、交換、出租或揭露予其他第三方。

七、Cookie 技術

本網站可能使用 Cookie 或類似技術，以改善網站功能及使用體驗。

您可透過瀏覽器設定管理或拒絕 Cookie，但部分功能可能因此受到影響。

八、資料安全

本公司採取合理之管理及技術措施，保護個人資料免於未經授權之存取、竄改、揭露或毀損。

九、政策更新

本公司保留隨時修改本隱私權政策之權利，更新後將公布於本網站，不另行個別通知。
